## 🤝 Contributing

We welcome contributions of all kinds!

### Ways to Contribute

| Type | Description |
|------|-------------|
| 🐛 Bug Reports | Found a bug? [Report it](https://github.com/niklasmarderx/ember/issues/new?template=bug_report.md) |
| 💡 Feature Requests | Have an idea? [Share it](https://github.com/niklasmarderx/ember/issues/new?template=feature_request.md) |
| 📝 Documentation | Improve docs, fix typos, write tutorials |
| 💻 Code | Fix bugs, implement features, optimize performance |
| 🌍 Translations | Help translate Ember to your language |
| 💬 Community | Help others in Discord/discussions |

### Getting Started

```bash
# Clone the repository
git clone https://github.com/niklasmarderx/ember.git
cd ember

# Build
cargo build

# Run tests
cargo test

# Create a branch
git checkout -b my-feature

# Make your changes and submit a PR!
```

**[Read the full Contributing Guide →](CONTRIBUTING.md)**

## 📜 Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for all.

### Our Standards

✅ **Do:**
- Be respectful and inclusive
- Use welcoming and kind language
- Accept constructive criticism
- Focus on what's best for the community
- Show empathy towards others

❌ **Don't:**
- Harass, discriminate, or intimidate
- Use sexualized language or imagery
- Troll or make personal attacks
- Publish private information

## 🆘 Getting Help

### Documentation
- [Getting Started](docs/src/getting-started/quickstart.md)
- [User Guide](docs/src/guide/cli.md)

### Support Channels
1. **Search first**: Check docs and existing issues
2. **Discord `#help`**: Quick questions and real-time help
3. **GitHub Discussions**: Detailed questions

### Response Times
- Discord: Usually within hours
