# Ember Roadmap

## Vision

Ember aims to be the most user-friendly, fast, and extensible AI agent framework written in Rust.

---

## Current Status (v0.1)

### Completed Features
- 7-crate modular architecture
- LLM abstraction (OpenAI, Anthropic, Ollama)
- Streaming responses with real-time output
- Built-in tools (Shell, Filesystem, Web, Browser)
- WASM plugin system
- SQLite storage with vector search
- CLI interface with TUI mode
- Web API (Axum)
- MCP (Model Context Protocol) support
- Plan/Act mode for complex tasks
- Checkpoints and undo functionality
- Local embeddings for RAG
- CI/CD pipeline
- Docker support
- One-liner installation
